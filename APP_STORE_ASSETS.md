## Apple App Store Requirements

### 1. App Icon (Required)

**Specifications:**
- Format: PNG (no transparency)
- Color Space: RGB
- Sizes required:
  - 1024x1024 (App Store)
  - 180x180 (iPhone @3x)
  - 120x120 (iPhone @2x)
  - 167x167 (iPad Pro @2x)
  - 152x152 (iPad @2x)
  - 76x76 (iPad)
  - 60x60 (iPhone notification @3x)
  - 40x40 (iPhone/iPad spotlight @2x)
  - 29x29 (Settings)

**Design Guidelines:**
- No rounded corners (iOS applies them automatically)
- Fill entire canvas
- Avoid text overlay
- Simple, recognizable design
- Should work well at small sizes

**Suggested Design:**
- Background: Gradient from indigo (#4F46E5) to purple (#9333EA)
- Icon: Stylized building or bridge symbol in white
- Modern, minimalist aesthetic

**Tools:**
- [appicon.co](https://appicon.co/) - Generate all sizes from 1024x1024
- [Figma](https://figma.com) - Design the icon
- [Sketch](https://sketch.com) - Design the icon

### 2. App Screenshots (Required)

**Required Sizes:**

**iPhone 6.7" (iPhone 14 Pro Max, 15 Pro Max)**
- Size: 1290 x 2796 pixels
- Minimum: 3 screenshots
- Maximum: 10 screenshots

**iPhone 6.5" (iPhone 11 Pro Max, XS Max)**
- Size: 1284 x 2778 pixels
- Minimum: 3 screenshots
- Maximum: 10 screenshots

**iPhone 5.5" (iPhone 8 Plus)**
- Size: 1242 x 2208 pixels
- Optional but recommended

**iPad Pro 12.9" (6th gen)**
- Size: 2048 x 2732 pixels
- Minimum: 3 screenshots (if supporting iPad)

**Suggested Screenshots:**
1. Landing page / login screen
2. Dashboard with properties
3. Place editor / link management
4. Profile/place page preview
5. Analytics dashboard
6. QR code generation

**Design Tips:**
- Use device frames for professional look
- Add descriptive text overlays
- Show key features
- Use consistent branding
- High quality, actual app screenshots

**Tools:**
- [Screenshot.rocks](https://screenshot.rocks/) - Add device frames
- [Rotato](https://rotato.app/) - 3D device mockups
- [Figma](https://figma.com) - Design screenshot layouts

### 3. App Preview Video (Optional but Recommended)

**Specifications:**
- Format: MP4 or MOV
- Duration: 15-30 seconds
- Same sizes as screenshots
- Resolution: Match screenshot dimensions

**Content Ideas:**
- Quick walkthrough of creating a place
- Demonstrating link management
- Showing QR code generation
- Analytics overview

## Google Play Store Requirements

### 1. App Icon (Required)

**Specifications:**
- Format: PNG (32-bit)
- Size: 512 x 512 pixels
- Max file size: 1MB
- No transparency or rounded corners (Play Store adds them)

**Design:**
- Same design as iOS icon
- Can use the 1024x1024 iOS icon and resize to 512x512

### 2. Feature Graphic (Required)

**Specifications:**
- Format: PNG or JPEG (24-bit)
- Size: 1024 x 500 pixels
- Max file size: 1MB

**Design Ideas:**
- App logo + tagline: "Your Hospitality Link Hub"
- Showcase key features with icons
- Use brand colors (indigo/purple gradient)
- Include phone mockup showing the app

**Tools:**
- [Canva](https://canva.com) - Feature graphic templates
- [Figma](https://figma.com) - Custom design

### 3. Screenshots (Required)

**Phone Screenshots (Minimum 2, Maximum 8)**
- Format: PNG or JPEG (24-bit)
- Minimum dimension: 320px
- Maximum dimension: 3840px
- Recommended: 1080 x 1920 pixels (Portrait) or 1920 x 1080 (Landscape)

**Tablet Screenshots (Optional, 7" and 10")**
- Same format as phone
- Recommended: 1200 x 1920 pixels

**Suggested Screenshots (Same as iOS):**
1. Dashboard view
2. Place editor
3. Link management
4. Profile preview
5. QR code generation
6. Analytics (if available)

### 4. Promo Video (Optional)

**Specifications:**
- YouTube URL
- Duration: 30 seconds to 2 minutes
- Should demonstrate app functionality

### Recommended Workflow

1. **Design App Icon**
   - Create 1024x1024 master in Figma/Sketch
   - Export for iOS (1024x1024)
   - Resize for Android (512x512)
   - Generate all sizes using appicon.co

2. **Take Screenshots**
   - Use simulator/emulator at required sizes
   - Capture key app screens
   - Add device frames using screenshot.rocks
   - Add text overlays if needed

3. **Create Feature Graphic (Android)**
   - Design in Canva or Figma
   - Use brand colors
   - Include app name and tagline
   - Export as 1024x500 PNG

4. **Prepare Text Content**
   - Write descriptions
   - Research keywords
   - Prepare URLs
   - Review character limits

5. **Review & Test**
   - Check all assets meet specifications
   - Test URLs are live
   - Review text for typos
   - Ensure consistency across platforms
